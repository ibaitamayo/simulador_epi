# Scalability Audit Plan

## Objective

Determine the maximum number of countries that can be simulated while maintaining an acceptable user experience.

## Metrics

- Simulation execution time
- Map refresh time
- Memory usage
- Scenario comparison responsiveness
- Export performance

## Country targets

- 50
- 75
- 100
- 125
- 150
- 200

## Success criteria

Target user experience:

- Simulation < 5 seconds
- Map refresh < 2 seconds
- No noticeable UI lag
- Stable memory consumption
